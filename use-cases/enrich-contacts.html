How Enrichment Works

1. Select Contacts to Enrich

From your contact list, select individuals you want to enrich. You can enrich one at a time or select multiple for bulk enrichment.

2. We Fetch the Data

WhoIMetWith queries professional data providers using the contact's email address. Within seconds, we pull available professional information.

3. Profile Gets Updated

The contact's profile is automatically updated with enriched data. You'll see their full name, current job title, company details, LinkedIn profile, and bio.

4. One Credit Per Enrichment

Each successful enrichment uses one credit. If we can't find data for an email address, you won't be charged.
